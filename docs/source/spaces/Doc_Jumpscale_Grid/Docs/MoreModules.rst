

Install more modules
====================


Execute some where in the bash shell:




.. code-block:: python

  #!bash
  cd /opt/code/jumpscale/jumpscale_core/utils
  python installAdditionalRepos.py


This will install quite some new modules.


* jumpscale_examples
* jumpscale_grid
* jumpscale_lib
* jumpscale_portal
* dfs_io_core
