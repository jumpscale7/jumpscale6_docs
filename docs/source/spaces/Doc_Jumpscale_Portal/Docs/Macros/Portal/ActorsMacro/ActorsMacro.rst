
Actors
######


View all actors. It accepts 2 parameters


* 'table': View actors in a table. This is true by default.
* 'bullets': View actors as a list (HTML 'ul'). This cancels the 'table'


Examples
********



.. code-block:: python

  \{\{actors\}\}





.. code-block:: python

  \{\{actors: bullets\}\}
